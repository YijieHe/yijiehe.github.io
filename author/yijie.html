For this assignment, the first step is to create a repository in github named yijiehe.io which is used for Github Page, and install Pelican using pip.

Then, I clone the github repository and start a pelican site using commmand "pelican-quickstart" in the repository.

Next, inside the content folder, write this blog into a rst file, and create website using the rst file. The files for the website are saved in the output folder.

After this, install ghp-import using pip, and push the changes in output folder on local git.

The final step is push the local git change to the remote connection with git remote push.
